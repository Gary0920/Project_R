# VMU客户参观接待流程

## 目的
规范客户参观 VMU 的接待流程，确保参观过程专业、高效、有仪式感，提升客户对公司整体专业度、服务质量的良好印象。

## 流程内容

### 流程步骤

1. **客户参观方案确认**：由项目负责人与客户负责人根据需求确认参观动线与顺序（会议室 - VMU 或反向），明确讲解重点。
2. **VMU 布置与样品展示**：设计组与 PMC 确认 LOGO 标识、样品摆放、指示标识及拍照点等布置。
3. **欢迎体验准备**：包括胸牌、欢迎资料、LED 屏欢迎词、茶歇布置。
4. **讲解与互动环节**：安排讲解顺序、讲解人员及互动体验流程，记录客户反馈。
5. **客户礼品准备**：确认并准备参观礼品并统一包装。
6. **交通与餐饮安排**：负责客户车辆接送、预订餐厅并安排菜单。

### 时间节点

| 时间节点 | 工作内容 |
| --- | --- |
| 参观前 1 周 | 确认参观方案、样品与讲解人安排 |
| 参观前 3 天 | 完成全部现场布置与讲解资料准备 |
| 参观当天 | 接送、引导、讲解、合影 |
| 参观后 1 天 | 客户反馈总结与归档 |

### 补充说明

- 整体流程应控制时间节奏，避免等待。
- 样品与讲解资料应尽量直观，标识清晰。
- 客户互动区可设置细节演示环节。
- 所有资料应拍照、统一归档备份。
