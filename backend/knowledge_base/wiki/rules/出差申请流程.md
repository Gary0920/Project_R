# 出差申请流程

## 目的

确保所有计划出差的安排被记录、审批，并实现部门协同与差旅管理规范。

## 核心规则

出差前必须完成该流程，如遇特殊情况可事后补提。

## 提交流程说明

- 所有计划出差的员工，需通过【出差申请表单】完成申请。
- 若多人一同出差，仅需一人提交表单，其他人员请在【同行人员】栏位中完整填写姓名。
- 如因临时安排未能提前提交，可在出差结束后尽快补填流程。

## 表单填写说明

填写路径：钉钉 → 工作台 → 首页的全部审批单里的搜索框 → 搜索【出差申请】

| 字段名称 | 填写说明 |
| --- | --- |
| 出差人姓名 | 手动填写 |
| 所属部门 | 下拉选择部门（必填） |
| 职务 | 输入职务 |
| 工作代理人 | 出差期间可协助处理事务的同事 |
| 同行人员 | 如有其他人员同行，请填写姓名 |
| 开始时间 | 选择出发时间 |
| 结束时间 | 选择返回时间 |
| 时长（小时） | 自动计算或手动填写，用于考勤统计 |
| 交通工具 | 下拉选择（公司车 / 自驾车 / 自乘车） |
| 备注 | 特殊时间或行程说明可填写 |
| 出差事由 | 简要说明出差目的 |
| 发送群组 | 可选择同步发送至相关部门群组 |

## 交通工具特别说明

| 交通工具 | 关联申请 |
| --- | --- |
| 公司车 | 需另行走 [[用车申请]] 流程 |
| 自驾车 | 需另行走 [[行政用车申请]] 流程 |
| 自乘车 | 无需额外申请流程 |

## 补充说明

如使用公司车或自驾车，务必提前走完对应用车流程，确保不会冲突、延误出行安排。
