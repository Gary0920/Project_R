# 盖章流程

## 目的

统一公司印章使用申请流程，确保用章合规、信息留存，提升盖章效率与管理透明度。

## 规则内容

1. 所有需要使用公司印章（如合同、公证件、授权书等）的文件，必须提前走钉钉【盖章申请】流程，审批通过后方可盖章。
2. 未走流程或未审批通过的文件，禁止盖章。
3. 提交流程时需上传需盖章文件电子档，并填写相关信息：盖章原因、文件名称、份数、文件类型、加盖章类别等。
4. 审批人顺序如下：直属支持人 → Shak → Nick，流程结束后自动抄送给人事相关人员（方美慧、黎凤娣、Tiffany、李明霞）。

## 流程步骤

| 步骤 | 操作内容 |
| --- | --- |
| 1 | 登录钉钉 → 工作台 → 表单 |
| 2 | 选择「广州分公司」 → 打开《盖章申请》表单 |
| 3 | 填写相关信息：经办人、盖章原因、文件名称、文件类别、加盖章类型、附件等 |
| 4 | 上传电子文件 |
| 5 | 设置审批流程：直属支持人 → Shak → Nick |
| 6 | 设置抄送人：美慧、风娉、Tiffany、明霞 |
| 7 | 点击提交，等待审批通过后，方可进行盖章操作 |

## 补充说明

- 【加盖章类型】可选择：合同章、公章、法人章等。
- 表单已设定默认审批路径，提交人只需选择好第一位支持人。
- 如遇紧急情况，请同步口头告知相关人员并在备注中说明。
