# 项目工期的把控

## 目的

为确保项目 Program（进度）可控，避免因客户确认延迟或变更导致生产中断、资源浪费及额外成本，特制定本规则，以明确关键节点的客户确认要求及沟通方式。

## 规则内容

### 一、订料前阶段的客户确认

1. 在订料前 1 周，项目负责人必须通过正式邮件向客户发出通知，明确说明：项目一旦进入下单/订料阶段，将无法中途停止或撤回
2. 若客户在邮件发出后 3 天内未回复，项目负责人须再次通过邮件重复确认该原则，以确保客户知悉并理解

### 二、铝材到厂前的 Program 把控

3. 当铝材预计在1周内到厂时，项目负责人需提前与客户沟通并强调：铝材一旦到厂，项目将必须进入生产阶段
4. 如客户在该阶段提出暂停或延迟生产的需求，需明确告知：可能产生仓储费用或相关额外成本

### 三、确认方式与默认原则

5. 所有涉及 Program 把控的重要确认，必须通过邮件进行正式沟通与确认
6. 在确认邮件中，项目负责人须明确说明：如客户对 Program 或相关安排有异议，需尽快以邮件形式提出；如客户在合理时间内未回复邮件，则视为默认同意相关安排
