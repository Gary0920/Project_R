# 装柜后的工作与沟通

## 目的
规范装柜后与客户、货代及内部的各项沟通与资料更新工作，确保出货信息透明、资料齐备、客户同步。

## 流程内容

### 装柜后工作步骤

1. **发送装柜照片给客户**：通过邮件向客户发送装柜照片，内容需包含：
   - 装柜日期、柜号、数量（如 3x40HQ）；
   - ETD（预计离港）与 ETA（预计到港）信息；
   - 使用 WeTransfer 附上照片链接，照片需包括铁架、货柜、散料（如有）等；
   - 参考邮件标题格式：Drift -- Loading pictures of L13-L17 window panels 3X40HQ containers。

2. **存档装柜资料**：包括：
   - 胜哥拍摄的散料/细节照片
   - 阿霞提供的铁架/柜内照片
   - Packing List（PL）

3. **更新 Logistic Table**：填写并更新最新发货信息，发送给 Matrak（如 William）。

4. **更新 Tracking Plan**：按格式填写并提交给客户。

5. **发送清关资料给货代**：按客户要求，将发票、箱单、装柜明细等整套清关所需资料发给客户指定货代。

### 时间节点

| 时间节点 | 要求 |
| --- | --- |
| 装柜当日 | 拍摄照片并收集资料 |
| 次日内 | 发客户邮件 + 更新 Log 表/Tracking 计划 + 发货代资料 |

### 补充说明
- 所有资料发送建议在装柜完成后 1 日内完成；
- 多个客户或项目同一批次出货时需分别命名、分别归档；
- 清关资料需提前确认客户收件邮箱与指定联系人。
