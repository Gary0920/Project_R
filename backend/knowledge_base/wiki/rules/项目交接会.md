# 项目交接会

## 目的
与中山工厂正式交接项目，确保项目资料完整传递，各部门了解任务内容与协作要求。

## 流程内容

### 1. 会议时间安排
中标后，根据项目进度安排交接会议，地点为中山工厂办公室。

### 2. 参会人员
| 角色 | 要求 |
| --- | --- |
| Nick | 可选 |
| Shak | 必须 |
| 中山团队（PMC、质检、采购、仓库、生产） | 必到 |
| 广州项目组（项目负责人、项目员、主设计、PMC GZ） | 必到 |
| 辅助设计 | 可选 |

### 3. 会议准备资料
- 材料清单（由设计提供）
- 已签字的 Shop Drawing（翻译版）
- 已签字物料清单（交接用）
- 建筑立面图、效果图
- 包装要求、项目工期（翻译版）
- 客户指定质检要求（如有）

### 4. 会议纪要与确认
- 项目员负责记录会议纪要；
- 会后由各部门签字确认纪要内容，归档保存。

### 补充说明
- 会前需提前三天将所有准备资料完成并预审；
- 所有签字版文件需扫描归档进共享盘。
