# 材料清单制造

## 目的
规范项目中材料清单从制作到后续更新的全过程，确保清单准确性与信息同步，提高加工效率。

## 流程内容

### 第一部分：材料清单制作与审核（项目主设计为主，加工主设计为辅）

| 步骤 | 说明 |
| --- | --- |
| 1 | 客户签字并确认了项目施工图后 |
| 1.1 | 由项目主设计主导并编制第一版材料清单 |
| 2 | 负责加工图的主设计审核并优化第一版材料清单 |
| 2.1 | 与项目主设计逐项确认 |
| 3 | 加工主设计审核并确认第一版材料清单 |
| 3.1 | 交由加工主设计编制 |
| 3.2 | 后期编制由加工主设计依据加工工艺再逐步调整和增加材料清单 |
| 4 | 加工阶段如加工设计调整或增加材料清单内容 |
| 4.1 | 需经过项目主设计审核与确认 |
| 4.2 | 涉及成本增加的需告知项目员审核成本，确认是否可以增加 |

### 第二部分：材料清单的更新与管理（加工主设计为主，项目主设计为辅）

| 步骤 | 说明 |
| --- | --- |
| 1 | 材料清单由加工主设管理 |
| 1.1 | 每周更新一次，并发相应的钉钉群通知大家 |
| 1.2 | 如本周版本未更新，需在钉钉群告知："本周材料清单无更新，沿用 R(X) 版本" |
| 2 | 材料清单的管理遵循图例中的原则进行更新调整，方便查询 |
| 3 | 所有涉及更新或更改的项目，需由更新或修改的人把信息截图放进材料清单旁边作为依据，方便审查 |
| 4 | 如涉及另外签字的施工图，项目主设计主导编制其他版材料清单（不与已提交生产的重复），由加工主设计确认后增加到已确认的材料清单中更新并通知 |

### 时间节点

| 时间节点 | 要求 |
| --- | --- |
| 施工图签字确认后 | 制作第一版材料清单 |
| 每周 | 更新材料清单并通知项目组 |
