# 铝型材明细下单规范

## 目的
明确铝型材在下单前的校对要点与标准，确保分类合理、编码准确、数量清晰，减少采购失误和沟通成本。

## 流程内容

| 分类 | 说明 |
| --- | --- |
| 是否开模 | 下单前需确认是否已开模；若订单中某款未确认开模，需单独拆单处理 |
| 归类排放 | 不同系统铝型材需分类排放；同系统的不同种类铝材也需分类排放 |
| 编码管理 | 每条型材下单都需有物料编码；找 ERP 专员詹吴霞生成编码 |
| 表面处理 | 表明颜色+材质要求，如"阳极氧化银白色"；确认色号 |
| 起订量标准 | 季华起订量 3 支（不收取上机费），如有 4000T 大机台时需同采购确认起订量；兴发起订量 20 支（不收取上机费），如不满 20 支（愿意支付上机费）不可定尺 L=6000mm |
| 供应商 | 不同供应商分单提料，例如季华和兴发模具不同 |
| 变更处理 | 若变更材质、颜色、长度，需重新编码 |
| 型材 | 型材只订不领，生产时单独出领料单 |

### 补充说明
- 本规则适用于所有铝型材订单（含系统料、结构料、自定义型材等），执行前应与设计组/采购组交叉确认；
- 建议建立铝型材下单 Checklist 供下单前内部交叉检查使用。
