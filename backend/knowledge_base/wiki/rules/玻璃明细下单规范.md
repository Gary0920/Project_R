# 玻璃明细下单规范

## 目的
规范玻璃下单资料提交要求，确保分类清晰、标注完整、工厂接收无误，减少错单与返工风险。

## 流程内容

| 分类 | 说明 |
| --- | --- |
| 确认供应商 | 下单前需同采购和供应商确认丝印模板的格式与尺寸 |
| 分类排放 | 不同种类玻璃需分类归档排放，如 GL01、GL02 等 |
| 排版图 | 要有玻璃排版图 |
| 标注与说明 | 附图需注明玻璃颜色、配置、胶深、澳标位置等关键信息 |
| 图纸要求 | PDF 图纸需带审批盖章签字，同时需附上 CAD 版本；严禁多页或少页 |
| 变更处理 | 若涉及图案、配置、尺寸变更，必须重新生成编码，不能延用原编码 |
| 独立拆单 | 不同供应商的玻璃需分开做单 |
| 数量 | 玻璃是订多少领多少；同步填写领用数量 |

### 补充说明
- 建议在排版图中增加编号索引，便于工厂快速查找、校对；
- 所有玻璃文件建议由项目组或设计组二次核查，确保图纸与排版逻辑一致；
- 若遇玻璃供应商变动，务必提前告知设计与 PMC，避免出错。
