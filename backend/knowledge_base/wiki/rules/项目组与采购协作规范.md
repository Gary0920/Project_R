# 项目组与采购协作规范

## 目的

提升与供应商沟通的专业性与一致性，避免项目组与采购部门沟通脱节、职责不清等问题，保障采购执行高效与外部合作稳定。

## 规则内容

### 1. 供应商谈判责任划分

- **项目组全程负责模式**：项目组需独立完成与供应商的所有沟通与谈判，最后将整理好的信息与结论交接给采购部下单
- **需采购部参与模式**：若需采购部参与，必须提前邀请采购部邓经理全程参与，拉入与供应商的大群/共享全部谈判信息。若前期已有沟通，必须整理总结清楚后再交接

严禁"沟通到一半谈不下去了才甩给采购部"的行为，此类行为已有多次被投诉。

### 2. 供应商沟通格式要求

- 所有对外沟通必须在供应商群内进行，禁止私聊或成员各自单线沟通
- 群内必须添加采购部邓经理为成员
- 重要沟通必须使用"总结性长信息"，说明背景、问题、结论，便于统一理解与记录

### 3. 执行方式总结

- 方式一：项目组全程谈妥 -> 总结 -> 交接采购
- 方式二：采购部全程参与 -> 协同推进 -> 采购执行

中途信息脱节、流程混乱将直接影响采购效率与合作质量。
