# 离职申请

## 目的

规范员工离职流程，确保交接顺利，手续齐全，减少因人员流动对工作造成的影响。

## 流程内容

1. 员工需在钉钉系统中提交【离职申请】。
2. 直属上级审批，确认离职日期（一般需提前30天申请，如提前或延后需协商）。
3. 离职前完成交接事项。
4. 到期后联系广州行政部门，办理正式离职手续。

## 表单填写项（钉钉）

| 字段名称 | 填写说明 |
| --- | --- |
| 姓名 | 填写本人姓名 |
| 部门 | 填写本人所在部门 |
| 职务 | 填写本人所任职位 |
| 入职日期 | 填写本人入职日期 |
| 最后工作日 | 由员工根据与主管协商填写 |
| 辞职原因 | 自由填写 |
| 辞职原因备注 | 进一步补充说明 |
| 抄送组 | 选择需要知悉的人员 |
| 审批人 | Shak |
| 流程提醒 | 建议提前30天发起申请，以便安排交接 |

## 补充说明

- 正式离职手续需联系广州行政负责同事。
- 如遇离职时间调整，建议提前沟通协商。
