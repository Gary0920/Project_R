# 发货前期准备指引

## 目的
为确保集中出货期项目的有序推进，规范项目组、设计组、PMC 组之间的协作流程，明确发货前的准备工作及关键执行要点，提升交付效率，减少因沟通不畅导致的问题。

## 流程内容

### 涉及项目（2025年4-7月）

| 项目编号 | 项目名称 | 发货计划 |
| --- | --- | --- |
| BG2412 | Italian Club | 6月底发货 |
| BG1138 | Windmere | 7月底发货 |
| BG0231 | Fortis Richmond | 暂定6月中发副框，待批图 |
| BG6001 | Sanura | 7月底发货 |
| BG2318 | Tapestry | 原定5月中出货，需再次确认 |
| BG0501 | Vela | 6月初发第一批副框，特批处理 |

### 项目组工作要求
- 尽快完成样品确认、图纸确认及交接资料准备；
- 协调客户、PMC、设计组，关注工期变化；
- 第一批货款到后才能下料；
- 所有特批需由 Shak 批准，Frank 把关；
- 客户批图后变更：不合理可拒绝，必须执行需再次走审批流程；
- 交接资料清晰完整，交接后不得私自修改。

### 设计组要求
- 曾胜统筹人员安排与工作量分配；
- 严格审核所有特批与变更内容。

### PMC 组要求
- 提前确认交期及发货计划；
- 负责落架及排柜沟通；
- 生产异常需及时反馈项目组。

### 协作提醒
- 集中出货期任务重，请保持良好沟通；
- 有困难及时联系 Fox 组协调支持。

### 补充说明
本规则适用于所有集中出货期项目，在未来类似场景中亦可参考执行。
