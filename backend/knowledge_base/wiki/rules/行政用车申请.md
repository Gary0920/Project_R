# 行政用车申请

## 目的

规范员工在因公出差时使用私人车辆的申请流程，确保相关车费可作为有效报销凭证。

## 核心规则

1. 本流程适用于因公外出时使用私人车辆的情况，如前往中山、客户地、工厂或其他城市。
2. 如需使用公司车辆，请走 [[用车申请]] 流程，不是行政用车申请，二者不能混淆。
3. 表单需在出差当日或前填写，报销时需与报销单一并提交。
4. 表单内容填写后，经直属主管审批后完成，无需其他额外审核流程。

## 表单内容（登记项）

| 项目名称 | 填写说明 |
| --- | --- |
| 用车事由 | 简要说明出差原因 |
| 出发地点 | 出发的具体位置 |
| 目的地（*） | 出差目的城市或地点（必填） |
| 返回地点 | 行程结束后返回的位置（可选） |
| 用车日期（*） | 实际使用车辆日期（必填） |
| 返回日期 | 实际返回日期 |
| 送回地点 | 车辆返回地点（如适用） |
| 人数（*） | 一同出行人数（必填） |
| 车型类别（*） | 小轿车/商务车等（必填） |
| 数量（辆）（*） | 所用车辆数（必填） |
| 是否安排司机 | 选择是否由公司安排司机 |
| 出发前公里数 | 保安填写，自驾车出发时读表数 |
| 回厂公里数 | 保安填写，返回后读表数 |
| 保安签字确认 | 保安手动签字确认 |
| 图片上传 | 可上传行程截图、油费单据等作为凭证 |
| 备注 | 其他需补充说明事项 |
| 抄送对象 | 可选择发送至相关同事群 |
| 审批流程 | 直属主管审批 |

## 文件路径

钉钉系统 → 表单 → 中山市博路德幕墙门窗系统有限公司 → 【行政用车申请】

## 补充说明

- 表单中如需添加多辆车，可点击"增加车辆"继续填写。
- 请勿误选为【行政用车申请】以外的其他用车表单。
- 保安签字说明：广州办公室的同事请找 Nicole 签字确认；中山的同事则找工厂保安人员签字。
