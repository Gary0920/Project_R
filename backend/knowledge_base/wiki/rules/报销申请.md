# 报销申请

## 目的

本流程旨在为员工提供清晰的费用报销指引，涵盖差旅费与一般费用两类，确保财务合规与高效执行。

## 报销类型

| 类型 | 类别说明 |
| --- | --- |
| 差旅费报销 | 工作出差中产生的费用，包括交通费、伙食费、住宿费 |
| 一般费用报销 | 除差旅费以外的所有费用报销 |

## 差旅费报销流程

可报销内容：交通、伙食、住宿

1. 提交时限：出差后1个月内必须完成报销
2. 需提前提交《出差申请单》，如未提交，可出差后补交，需随报销单一并上交
3. 自驾出行：需提交《行政用车申请》，可事后补交，但同样需随报销资料一并提交

## 一般费用报销流程

4. 所有非差旅类报销按此流程操作
5. 报销项目如超过2项，需附汇总表以便审批与财务清晰查看

## 统一遵循规则

6. 所有报销须关联具体项目预算，不得随意上报
7. 报销需由本人提交，不可代他人申请
8. 如使用"替票"，须同时满足以下三项：

### 替票规则

| 条件 | 说明 |
| --- | --- |
| 文件中不得出现"替票"字样 | 不论电子或纸质文件均适用 |
| 替票不得跨年使用 | 仅限当年报销使用 |
| 必须有交易记录凭证 | 证明费用真实发生 |

## 实际操作流程

| 步骤 | 动作 | 说明 |
| --- | --- | --- |
| 1 | 钉钉提交申请 | 填写对应的报销单 |
| 2 | 上级审批 | 直属主管审批 |
| 3 | 打印报销 + 附原始发票 | |
| 4 | 提交纸质资料 | 交至中山财务部门归档报销 |
