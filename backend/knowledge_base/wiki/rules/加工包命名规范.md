# 加工包命名规范

## 目的

确保项目资料命名一致，便于后续查找、对接与归档，避免因命名混乱导致沟通或版本错误。

## 规则内容

### 命名格式结构

`xxxxxx BGxxxx [JG-xx 范围] 加工文件种类`

- `xxxxxx`：为日期（例：220402）
- `BGxxxx`：为项目编号
- `[JG-xx 范围]`：为具体分区和楼层范围，或 [PMU]
- `加工文件种类`：如全套、机加和组装图、生产任务表格包、领料单等

### 示例

| 命名 | 说明 |
| --- | --- |
| 220402 BG0111 [JG-13 L15-18] 全套 | 日-项目-分区楼层-全套 |
| 220515 BG0505 [JG-18 L24-30] 机加和组装图 | 日-项目-分区楼层-机加组装 |
| 220823 BG8001 [JG-02 L02-03] 生产任务表格包 | 日-项目-分区楼层-生产任务 |
| 220510 BG8001 [PMU] 领料单 | 日-项目-PMU-领料单 |
