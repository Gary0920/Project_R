# VMU 标准作业流程

## 目的
规范 VMU 样板全流程操作，从内部确认 SOW、客户确认图纸、生产加工、客户参观验收到复盘归档，确保每个环节有标准可依、责任明确。

## 流程内容

### VMU 全流程总表

| 阶段 | 标准动作 | 固定标准 | 注意事项 |
| --- | --- | --- | --- |
| 1. 内部确认 SOW | 项目组根据中标方案整理 VMU 展示范围，内部确认后发客户确认 | 使用统一《SOW确认表》模板；抄送 Design、PMC、Procurement；SOW 须标明不含范围 | 避免提前发未核对版本；文件页脚注明版本日期 |
| 2. 客户确认后出施工图 | 设计组绘制 VMU 施工图，项目组追踪客户审批进度 | 出图前冻结展示范围；用统一图框；客户批准后方可下发 | 多次修改易延误，应一次性确认；图纸版本须留档 |
| 3. 制作 VMU 样板布局图 | 设计师编制展示布局图（总览区、系统区、体验区），与客户确认 | 含导视、编号、动线；客户确认后归档 | 动线混乱会影响体验，提前彩排验证 |
| 4. 制作加工单与审图交接 | 设计组出工程料单并交接给 PMC 与工厂 | 使用统一工程料单模板；PMC 确认数量与规格；Design 参与审图 | 审图记录不完整易造成生产误差，须有签字记录 |
| 5. 技术交底会议 | 生产前召开会议，明确系统细节、安装方法、关键尺寸 | FOX 出会议纪要模板；会议录音备份；会后发送至群组 | 会后未跟进改图将导致信息断层 |
| 6. 生产加工与巡检 | PMC 追踪生产进度并记录关键工序；Project 和 Design 阶段性巡检 | 拍照留存并归档；缺陷点当日汇报；驻厂跟进 | 无巡检或照片会导致后期问题无凭据 |
| 7. 到厂前准备与彩排 | 所有人提前 1 天到厂，检查样品、布线、讲解设备 | 使用彩排检查表；麦克风、音响、照明测试；走流程一次 | 常见问题：音量不足、讲解节奏不协调 |
| 8. 现场布置与安全检查 | 布置展示区、会议区、动线指示、铭牌、品牌背景 | 标注"完成面""楼板线"；木地板拼缝加固；走道 >= 1200mm | 易忽视安全边缘与电源线路，须专人检查 |
| 9. 客户参观与讲解 | 舞狮迎宾 - 主持介绍 - 总讲 - 分区讲解 - 客户体验 | 分组陪同：每客户配一位技术/项目；统一工装；讲解用卡片提要 | 英语口径准备不足、语速过快影响理解 |
| 10. 会议室确认材料与问题 | 参观结束进入会议室确认材料、尺寸、系统与变更 | 使用统一《VMU Notes & Issue Log》逐条复核 | 若无模板记录，会导致后期遗漏 |
| 11. 客户签字与验收 | 客户确认无误后签字或邮件确认；拍照归档 | 所有确认点留痕（签字、邮件截图）；由项目组整理 | 若无书面记录，后期争议无法追溯 |
| 12. 复盘与归档 | 1 天内收集问题清单；7 天内复盘并更新 SOP | 复盘模板含：亮点、改进点、责任人、截止日；素材归档 | 延迟复盘易失去关键信息；预算复盘不可遗漏 |

| SOW 确认 | SOW 确认表/邮件 | Project | T-8~7W |

| 布局确认 | VMU Layout Plan | Design | T-6~5W |
| 工程料单 | 工程料单+审图表 | Design+PMC | T-5~4W |
| 技术交底 | 会议纪要 | FOX+PMC | T-4W |
| 生产巡检 | 照片与巡检记录 | PMC+Project | T-3~2W |
| 彩排 | 检查表、走流程 | 全员 | T-1W |
| 现场布置 | 安全检查表、铭牌安装 | PMC+Project | T-1~D-day |
| 客户参观 | 讲解卡、导览图、拍摄记录 | Project+Design | D-day |
| 会议室确认 | Issue Log、照片 | Project | D-day |
| 客户验收 | 签字表/邮件截图 | Project | D+1 |
| 复盘归档 | 复盘纪要、素材包 | FOX | D+7 |
