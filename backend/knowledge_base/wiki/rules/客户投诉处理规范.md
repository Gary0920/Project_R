# 客户投诉处理规范

## 目的

确保客户投诉（客诉）处理闭环、编号清晰、责任明确，提高处理效率与结果可追溯性。

## 规则内容

### 客诉编号命名规范

每个项目的客诉需依次编号，格式示例：`BG1133-01`、`BG1133-02`，依发生顺序顺延编号。

### 审批流程

所有客诉事项必须经以下两位责任人审批：

| 负责人 | 职责 |
| --- | --- |
| PMC组 -- 祝东东 | 负责安排打单、领料、打包、寄送等执行工作 |
| 设计组 -- 曾胜 | 负责审查是否涉及重新出图或技术调整 |

未经审批的客诉，不得下单、不得打包寄送。

## 补充说明

- 所有审批建议通过钉钉或邮件留痕
- 客诉资料应与"客户投诉登记表"同步编号，方便后续归档与追踪
