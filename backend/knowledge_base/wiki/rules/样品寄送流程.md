# 样品寄送流程

## 目的
规范样品寄送操作流程，避免中转混乱，提升样品交付效率和客户满意度。

## 流程内容

### 1. 项目组全权负责样品寄送流程
- 项目组使用 DHL 异地取件服务，自行完成样品准备、贴标签并下单发货。

### 2. 不允许将样品中转寄往中山
- 所有样品（含客户签字确认样品）统一直寄广州办公室；
- PMC 组有权将寄往中山的样品退回。

### 3. 非工厂准备样品的处理
- 项目组需自行准备并完成寄送，不得依赖工厂协助。

### 4. 客户签字样品如确需寄至工厂
- 需由 PMC 组签发指令，交由工厂采购和品质部门统一处理。

### 时间节点

| 时间节点 | 工作内容 |
| --- | --- |
| 样品准备完成当日 | 项目组发货并记录台账 |
| 1 个工作日内 | 广州办公室签收确认 |

### 补充说明
- 所有样品发出记录应同步记录在项目共享台账中；
- 项目员需提前与广州办公室确认收件人信息及收货时间；
- 如涉及海外寄送，需提前确认是否满足清关资料要求。
