# 借款申请

## 目的

为规范公司内部借款流程，确保借款事项合规、记录清晰、审批顺畅，并依据《员工借款管理制度》执行。

## 适用范围

适用于员工因公司业务需要申请临时借款的情况。个人用途的小额借款已不再支持。

## 操作路径

钉钉 → 工作台 → 借款申请 → 填写并提交 → 系统自动发起审批

## 表单填写说明

| 字段名称 | 填写说明 |
| --- | --- |
| 借款人（必填） | 申请人本人 |
| 借款事由（必填） | 简要说明借款用途及背景 |
| 申请金额（必填） | 需借款的总金额 |
| 币种（必填） | 默认人民币，如有特殊请备注说明 |
| 收款人账号（必填） | 本人工资卡账号（用于转账） |
| 上次借款情况 | 是否仍有未结清借款（如有请填写下方"清偿情况"） |
| 清偿情况 | 若有上次借款未结清，说明还款进度及安排 |
| 附件 | 可上传预算表、借款用途相关说明 |
| 备注 | 其他补充说明 |
| 会计复核 | 由会计填写 |
| 财务负责人 | 由财务负责人填写 |
| 出纳 | 由出纳填写 |
| 取款人 | 由取款人填写 |

## 流程说明

提交表单后，系统会自动发起审批流程。审批通过后，由财务根据填写的账号安排打款。

## 补充说明

- 请确保借款仅用于公司业务，不得作为个人资金使用。
- 借款审批通过后，如无特殊情况应尽快安排使用与还款，并配合后续财务结算。
