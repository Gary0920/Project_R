# 澳洲客户参观工厂

## 目的

规范澳洲客户来工厂参观时项目组的准备与接待流程，确保流程高效、专业、有序，提升客户体验。

## 规则内容

### 前期通知与准备

1. 通知投标组（Marketing Team）：一收到客户要参观工厂的信息，项目组需第一时间通知 Shak、Howard 等投标组相关人员
2. 项目组全方位准备：
   - 安排司机：确定接客户的时间、地点，走钉钉用车流程
   - 陪同人员确认：确认由谁陪同（如Shak、Jennifer、Yap等）
   - 会议室准备：确认是否使用一楼会议室，提前做好布置
   - 安排客户用餐：根据Shak建议确认客户用餐地点、人数，完成订位
   - 准备会议事项资料：确认需与客户过的事项，包括图纸、样板、文件等
   - 安排返程：确认送客户返程的司机及车辆

### 工厂协调准备

3. 通知工厂相关负责人：COO、生产经理、广州PMC需提前知晓
4. 安排工厂入口欢迎牌，包括客户公司Logo（请联系刘总工制作）

## 时间节点

| 时间节点 | 要求 |
| --- | --- |
| 接待前2天 | 通知投标组与工厂、完成用车与会议室安排 |
| 接待前1天 | 样板准备、会议资料打印、餐饮确认 |
| 接待当天 | 提前30分钟完成现场布置与人员就位 |
