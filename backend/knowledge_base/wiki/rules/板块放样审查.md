# 板块放样审查

## 目的
确保在加工生产前，提交的 CAD 模板放样资料完整、准确，避免返工和尺寸误差。

## 流程内容

### 1. CAD 图纸放样要求

以下内容需提交进行审查：

| 项目 | 内容说明 |
| --- | --- |
| 每一种类型板块 | 需要横竖剖节点放样 |
| 每一种连接方式 | 需要横竖剖节点放样 |
| 每一种节点 | 需要横竖剖节点放样 |

审查重点提示：放样尺寸与加工细目尺寸互相对应，避免比例错误或加工偏差。

### 2. 三维建模要求
- 暂不强制执行；
- 三维建模目前仅建议试行，不作为本阶段必须提交内容。

### 补充说明
此准则适用于所有类型的板块深化放样审核；推荐结合项目复杂程度判断是否引导团队试行三维建模。
