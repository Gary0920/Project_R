# 工作沟通规则

## 目的
建立公司内部统一沟通标准，规范语言、工具与沟通方式，提升信息传递效率，减少误解与重复沟通成本，营造高效、专业的办公氛围。

## 规则内容

### 办公室英语使用规则
为提高英语思维和客户沟通效率，办公室内需使用英文进行沟通。

| 部门 | 语言要求 |
| --- | --- |
| 项目组、投标组、销售组 | 必须使用英文 |
| 设计组 | 鼓励使用英文 |

若遇复杂情况可使用中文讨论，但必须同步英文记录总结。

### 手机使用规则
- 上班期间手机统一放置于公司提供的手机袋中
- 若需使用手机，请离开工作位，至公共区域处理

### 微信禁令

| 行为 | 处罚 |
| --- | --- |
| 使用微信进行工作沟通 | 罚款200元 |
| 建立微信工作群 | 群主罚款200元 |
| 在电脑端登录微信 | 罚款200元 |

所有工作讨论必须使用公司官方工具（如钉钉、Outlook 等）

### 长信息沟通原则
所有沟通需整理成逻辑清晰的长信息，包括背景说明、需求细节与附件，避免碎片化沟通。

**为什么要长信息？**
- 信息清楚、理解一致
- 减少反复沟通
- 提高整体效率

### 与供应商沟通规范

| 要求 | 细则 |
| --- | --- |
| 沟通渠道 | 仅限钉钉供应商大群 |
| 沟通格式 | 使用长信息形式 |
| 阶段更新 | 在群内进行阶段性总结 |
| 沟通结论 | 明确结论，或由邓经理协助决策沟通 |

## 补充说明
- 所有员工需每季度复盘一次沟通问题，可作为绩效参考项之一
- 长信息模版可向 Fox 小组获取
- 若因违反工具规定导致信息遗漏或错误，将由责任人承担
