# 项目启动会（中标统筹会）

## 目的
中标后召开项目启动会议，统一项目背景信息与关键时间节点，确保所有关键岗位理解项目要点，会议以简明介绍为主，不涉及讨论。

## 流程内容

### 召开时机
中标后 3 周内由项目组组织召开项目启动会。

### 会议目的
- 项目介绍
- 关键时间节点说明（如 VMU/PMU/Bulk 等）
- 强调：仅做介绍，不讨论技术细节

### 参会人员安排
- **管理层**：Ying GM（必须）
- **中山团队**：PMC（必须），品质、采购、仓库、生产（可选）
- **广州团队**：项目负责人、项目员、广州 PMC、Tiffany（必须）；设计、Shak（可选）
- Nick 为可选参会人

### 会议准备文件
- 精简版 Info Pack（只介绍不讨论）
- 项目信息：建筑图、效果图
- 项目特点说明（如设计难度、材料特殊性）
- Doris 提供的项目里程碑时间表：

| 内容 |
| --- |
| 1. 开模时间 |
| 2. VMU 下采购单时间 |
| 3. VMU 下加工单时间 |
| 4. PMU 下采购单时间 |
| 5. PMU 下加工单时间 |
| 6. 大货下单时间 |
| 7. 第一批发货时间 |

### 会议地点
中山工厂办公室 或 广州办公室

### 补充说明
- 会议须由项目员主导，提前 1 周发出会议邀请及资料包；
- 参会确认名单应归档备查。
