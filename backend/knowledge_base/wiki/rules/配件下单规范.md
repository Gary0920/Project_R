# 配件下单规范

## 目的
规范五金配件类物料下单流程与图纸资料要求，确保分类清晰、信息完整、责任明确，避免错单与工厂混淆。

## 流程内容

| 分类 | 要求说明 |
| --- | --- |
| 材质及规格标注 | 如非标准件，需同采购确认清楚规格及质量要求；同时确认起订量（MOQ） |
| 分类摆放 | 不同窗型五金配件要分开归类摆放（前提为同一供应商），例如：上悬窗的配件放在一起，推拉门的放在一起 |
| 非金属类另做单 | 不同物料辅材分开做单，如辅件较杂最多允许混做 3 种 |
| 使用位置说明 | 胶条、螺丝、辅件只订不领，生产时单独出领料单；胶条需注明具体使用位置；钢板钢件需标明位置或提供排版图；铝板、镀锌板、钢件订多少领多少，同步填写领用数量；滑轮、执手、风撑等五金需同采购确认单位"个"/"套"/"对" |
| 加工件 | 加工件需要附图纸：PDF 加工图需要审核盖章签字；提供 CAD 版一起打包下发；核查图纸页码，不允许多或少 |
| 变更处理 | 变更色号、规格、材质等信息均需要重新编码 |

### 补充说明
- 所有配件请提前在项目群沟通确认，避免因规格不明引起工厂延误。
