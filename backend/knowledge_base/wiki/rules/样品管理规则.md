# 样品管理规则

## 目的
规范从索要、收集、确认到存放的整个样品生命周期管理，确保 BFI 项目图文相符、签样齐全且存档合规。

## 流程内容

### 1. 样品索要与收集路径

| 类别 | 索要路径 |
| --- | --- |
| 铝型材与铝板 | 项目组发邮件找采购打样；采购收到供应商实物后统一交由 PMC 阿燕寄送给项目组 |
| 玻璃样品 | 项目组直接联系供应商（如新华强、南亮、信义、南星等） |
| 常规五金（BFI五金） | 项目组向设计下样单，货到工厂后项目设计出领料单，由广州 PMC 领出并制作五金展板，展板直接从中山工厂寄出给客户 |
| 非常规五金 | 项目组直接联系供应商（如坚朗、希美克、希洛克）获取实物并提交客户签样 |
| 辅材（防鸟网&防撞贴） | 常规用黑色不锈钢款，由广州 PMC 领出，通常从中山工厂寄给客户签样 |
| 密封胶&结构胶 | 无需实物样板 |

### 2. 样品数量要求

| 类别 | 数量要求 |
| --- | --- |
| 铝型材/铝板 | 至少需要 6 个签样的样品 |
| 玻璃样品 | 客户确认前每款需 2 个实物样板；签样后需找供应商补 4 个，总共至少 6 个签样的样品（用途：工厂质检、采购留底、密封胶测试） |
| 五金样品 | 通常为 3 套（一套送样、一套 PMC 留底、一套广州设计留底）；若项目需要 PMU/VMU/AMU 等需额外考虑备料数量 |
| 执手 | 需 3 套签样（不含 PMU 等特殊需求用量） |

### 3. 寄送与沟通
- 寄送责任原则上由项目组寄出，不由 PMC 负责；
- 五金展板由广州 PMC 制作并寄出，项目组须提前填写广州邮寄地址并告知 PMC。

### 4. 样品存放规则

**签样前：**
- 存放于广州办公室；
- 小体积样品（型材/五金）放入项目样品盒子；
- 大体积样品（玻璃等）放到广州办公室外的展架上。

**签样后：**
- 统一移交至 PMC 部门存放和保管；
- 项目组在实物上签字后邮寄给 PMC 统一存放，广州办公室不存放实物样板。
