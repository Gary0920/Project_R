# 外币付款流程

## 目的

规范在钉钉系统提交外币付款申请时的信息填写标准，确保收款方信息准确无误，提升财务支付效率。

## 规则内容

### 提交要求

- 提交平台：钉钉提交"付款申请单"
- 付款内容：格式为【项目编号 + 项目名称 / 请款内容】
- 金额：填写此次请款的外币金额
- 收款方：填写正确的收款方全称
- 开户银行：必须写清楚以下信息：Bank, BSB, Account, ACN, ABN, Swift Code
- 备注：写明支付币种；其它需注意信息

### 附件清单

- Subcontract Payment Register（分包付款登记表）
- Invoice（发票）
- Contract（合同）

### 付款水单存储规范

- 存储位置：`\\BOLU\09 Register\Payment Record\07 付款水单`
- 命名规范：`YYMMDD (付款日期) + 空格 + 项目编号 + [币种及金额]`
- 示例：`251028 BG1133[AUD 20,000]`

## 补充说明

- 所有必填项及附件缺失将导致流程被退回
- 请务必核对 Swift Code 和 BSB 信息的准确性，以防结汇失败
