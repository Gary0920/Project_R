# 开模流程

## 目的
规范从客户确认、图纸反馈到 OA 申请的整个开模流程，确保图纸准确性、审批齐全性和开模合规性。

## 流程内容

### 1. 开模前确认（广州内部主导，Frank 和项目组决策）
- DIP 图纸需由工程总监审批签字；
- 客户需通过邮件或签字确认同意开模。

### 2. 开模操作步骤
1. **发图**：CAD 图纸完成后由项目组发送给开模供应商。
2. **供应商反馈**：供应商会提出问题，项目组需及时沟通并修正，最终确认图纸后方可开模。
3. **准备附件**：包含已签字图纸 + 项目员汇总的本次开模费用明细表。
4. **提交 OA 流程**：
   - 审批人：项目组负责人、设计主管、预算部经理、工程副总、郑总；
   - 抄送人：项目员、双方 PMC、采购部主管。

### 补充说明
- 若客户未签字或未邮件确认，不得提交 OA 申请；
- 所有文件需统一上传至项目文件夹归档。
