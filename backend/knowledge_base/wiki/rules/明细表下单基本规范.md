# 明细表下单基本规范

## 目的
规范所有项目的材料提料、批次管理、图纸对应等下单流程，避免信息混乱、单号重复、批次混乱等情况发生，提升工作效率与交付准确性。

## 流程内容

| 分类 | 规则说明 |
| --- | --- |
| 提交流程要求 | 所有提料单必须发送至"提料下单群"，不允许在群外或跨项目群体私发 |
| 批次提料规范 | 严格按照提料分区及生产批次下单；每批提料需满足相应批次的生产领用要求；如发现提料数量不足或漏下，单独针对漏下补料重新下单；不允许私自决定混入下个批次订单中 |
| 图纸版本与下单一致性 | 明确图纸与加工批次一一对应，配件特殊的可根据需求灵活调整 |
| 单号唯一性要求 | 单号不允许重复 |
| 注明信息要求 | 必须注明楼层、系统、具体位置 |
| 变更命名格式 | 如变更内容需标注格式为 BG0111/101A，变更位置用粗虚线标注，便于识别 |
| 不同供应商分开做单 | 不同供应商不能混合做单，必须拆分下单 |
| 空白备注说明 | 空白位置注明"以下空白" |

### 补充说明
- 建议所有项目使用统一的提料单模板并进行编号管理；
- 如涉及特殊工艺结构，应在备注中额外说明用途与分类。
