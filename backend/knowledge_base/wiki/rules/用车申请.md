# 用车申请

## 目的

当员工因公需借用公司车辆时，必须走此【用车申请】流程，经审批通过后方可用车，确保公司资源合理调配、行程留痕、责任可追溯。

## 适用范围

- 适用于所有前往中山或其他城市出差、拜访客户、前往项目地或工厂等业务相关出行。
- 本表单不适用于行政类用车申请，请勿误选。如因公出差时使用私人车辆的申请流程，请走 [[行政用车申请]] 流程。

## 登记路径

登录公司系统 → 进入「广州分公司」分类 → 点击表单「用车申请」提交填写

## 填表内容

| 字段名称 | 填写说明 |
| --- | --- |
| 用车事由 | 简要描述此次用车目的，如"项目A外出测量" |
| 出发地点 | 可填写公司或其他具体出发点 |
| 目的地 | 请准确填写主要目的地（必填） |
| 返回地点 | 若去程与返程不同，请在此说明 |
| 用车日期 | 用车当天的具体日期（必填） |
| 送回日期 | 非当天返回请填写 |
| 送回地点 | 车辆预计返还位置 |
| 人数 | 同行人员人数，便于安排车辆（必填） |
| 车辆类型 | 如商务车、皮卡等，请根据实际需求选择（必填） |
| 数量（辆） | 需用几辆车就填几辆，若需多种车型，请点击"增加车辆"填写多条记录（必填） |
| 备注 | 任何特殊需求、时间说明等 |
| 图片 | 如需上传路线图、任务单等最多可上传9张 |
| 发送到群 | 可选择发送流程通知到指定群组 |

## 注意事项

- 用车记录将作为后续财务报销的辅助凭证。
- 若需行政车辆请走 [[行政用车申请]] 流程。
