# 样品登记表与玻璃最大面积计算

> 本文为培训/经验分享资料，不等同于公司正式制度。

## 一句话总结
统一项目样品登记表的通用格式与核心字段，并重点讲解玻璃最大许用面积的计算规则、安全玻璃标识识别以及热裂风险，旨在建立标准化的送样管理与玻璃选型知识库。

## 核心内容

### 样品登记表核心字段
Sample No. / Folder No. / Prepared by / Warranty（质保年限，必须包含）/ Physical Sample / Performance Data / Date Submitted。

### 送样策略
- 涉及颜色确认的批次（如玻璃、粉末喷涂铝型材）必须**组合成套**送出，避免分开发送导致客户无法决策或样板丢失。
- 送样文件夹结构标准化：按材料类型分类，预存常用供应商的Company Profile和Warranty Statement。

### 玻璃最大面积计算规则
- **计算基准**：以**单片玻璃**的厚度为准，而非中空或夹胶玻璃的总厚度。
- **常规限制**：6mm钢化玻璃最大面积通常不超过8平方米（如2m x 4m）。
- **特殊情况放宽**：若玻璃两面均可接触（如阳台玻璃），最大面积可乘以系数1.5。
- **夹胶玻璃厚度**：5mm+5mm夹胶视为10mm厚，但4mm+4mm在工厂实际不可行应避免设计。

### 玻璃强度等级与热裂风险
- 强度对比：全钢化(FT) > 半钢化(HS) > 普通退火玻璃(Annealed)。
- 普通退火玻璃边缘强度极低，工厂操作易碎且存在热裂风险，项目中应避免使用。
- 半钢化玻璃强度约为全钢化的一半，是平衡强度与平整度的常用选择。

### 安全玻璃标识识别
- 合规标识必须包含：AS/NZS 2208、玻璃类型（如T代表钢化）、等级（Grade A）、厚度（如6mm）。
- 中空玻璃（IGU）若为安全玻璃，每片玻璃上都应有独立标识，不能只印一片。

### 避坑要点
- 不要等所有材料到齐才送样，颜色相关材料到齐后应立即组合送审。
- 遇到超大玻璃（如3.6m x 2m）必须手动复核，若6mm不够应升级为8mm或10mm。
- 除非有特殊光学要求，否则不要使用普通退火玻璃，其加工破损率极高且存在自爆风险。
- 不要设计4mm+4mm的夹胶玻璃，主流供应商无法生产。
