# 项目状态标签

## 目的

定义项目规则库中每条规则的执行状态，帮助团队成员快速识别规则的当前效力。

## 状态列表

| Status | 中文状态名 | 说明 |
| --- | --- | --- |
| Active | 已生效 | 当前正式执行的规则，所有成员需遵守，适用于日常参考 |
| Draft | 起草中 | 尚在内部讨论、初步拟定中的规则，尚未定稿 |
| Under Review | 审核中 | 内容已形成，但正由上级或跨部门审核确认 |
| Deprecated | 已废止 | 曾使用但已废止，或被其他规则替代，保留历史记录 |
