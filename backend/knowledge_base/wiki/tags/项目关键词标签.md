# 项目关键词标签

## 目的

定义项目规则库中使用的关键词标签体系，用于统一标记、分类和筛选所有项目规则与流程文档。

## 标签列表

| Tag | 中文标签名 | 说明 |
| --- | --- | --- |
| Financial | 财务 | 涵盖与项目相关的所有财务内容，包括预算编制、内外报价、成本控制、开票、税务以及预算变更的审批流程等 |
| Material | 材料 | 与项目中使用的材料相关内容，包括玻璃、色粉、型材、五金、胶条等，涉及材料规格、选型标准及质量控制等 |
| System Design | 系统设计 | 系统性设计相关内容，比如推拉门系统、五金选择及工程相关问题的处理，也涵盖系统与组件的选择和定制化 |
| MU | 样板测试 | 所有样板测试相关内容，包括 AMU、VMU、PMU 的流程、标准、测试结果与审批所需文件 |
| VO | 变更 | 各阶段因内部或外部原因产生的变更单，包括设计变更、范围调整、现场条件变化等 |
| Subbies | 分包管理 | 与分包商或供应商相关的内容，包括分包方选择、协调、管理及相关文件，涵盖合同管理、绩效监控与外部合作方沟通 |
| QC | 质检 | 质量控制流程，包括各阶段的质检程序、检查清单与质量保障措施 |
| Templates | 模板 | 包括公司各部门使用的标准模板、命名规则、资料清单等内容，用于统一规范与提升专业性 |
| Contract | 合同管理 | 所有合同相关文件及管理内容，包括与客户、分包商、外部合作伙伴的协议 |
| Defect | 缺陷处理 | 从采购至完工阶段出现的所有缺陷问题，包括材料、产品或性能方面的不良或不合格项 |
| HR | 人力资源&任务分配 | 与人力资源相关的内容，包括入职、任务分配、职责说明与组织结构 |
| Communication | 沟通 | 公司内外沟通的方式、工具与注意事项 |
| Tools | 工具 | 与项目执行或日常操作中所使用工具相关的内容，包括软件平台、追踪系统、设备操作与使用指南等 |
| Hand Over | 交接 | 项目各阶段或团队之间的重要交接，例如投标组、项目组、设计组、PMC、工厂、外包之间的交接内容 |
