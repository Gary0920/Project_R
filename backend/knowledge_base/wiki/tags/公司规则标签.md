# 公司规则标签

## 标签分类说明

| 标签（Tag） | 中文标签名 | 说明 |
| --- | --- | --- |
| **General** | 通用 | 适用于全公司、跨部门通用的制度，例如命名规范、书面化原则、内部沟通方式等 |
| **HR** | 人事 | 涵盖所有与人事相关的制度，如打卡制度、培训安排、岗位职责、员工福利与健康计划等 |
| **Finance** | 财务 | 聚焦公司内部的财务制度，如利润分配、费用报销流程、财务审批等 |
| **Admin** | 行政 | 涉及日常行政运营的规则，如会议管理、办公设备使用、客户来访时员工着装与办公环境等 |
| **IT** | 信息技术 | 涵盖所有信息技术相关的制度，包括邮箱使用、服务器文件管理、电脑配置、资产登记、Notion权限等 |
| **Legal** | 法务 | 涉及保密协议、资料外发审批、公司合规管理等方面，保障公司权益 |
| **Operation** | 运营 | 关于内部运营结构或流程的规则，如团队职责分工、跨组协调流程等 |
