# 项目时间线标签

## 目的

定义项目全生命周期的阶段标签，用于标记每条规则适用的时间节点和阶段，支持按阶段筛选和检索相关规则。

## 阶段列表

| TimeLine | 中文标签名 | 说明 |
| --- | --- | --- |
| Project Info | 项目信息 | 项目的基础信息与背景介绍，包括建筑图、结构图等客户提供的核心文件资料 |
| Tender | 投标阶段 | 项目投标阶段的内容，包括报价、技术资料、方案选择及相关前期沟通记录 |
| Contract Award | 中标阶段 | 中标阶段涉及的内容，包括合同确认、澄清沟通与初期启动计划 |
| Sample | 样品管理 | 所有与样品相关的内容，包括样品清单、送样确认文件、检测报告、照片与评语记录 |
| SD | 施工图 | 项目早期的施工图设计，用于客户确认及与顾问单位协调使用 |
| Engineering | 工程技术 | 项目中所需的工程计算、技术验证及结构分析，确保符合规范并保障性能 |
| Fab Design | 生产加工设计 | 为工厂加工准备的图纸设计内容，包括加工图、切割计划与生产细节 |
| Procurement | 采购 | 采购相关的所有工作，包括材料询价、报价、下单及与供应商沟通内容 |
| Fabrication | 生产加工 | 工厂的生产加工流程，包括排产计划、加工记录及产出跟踪等内容 |
| Logistics | 物流 | 与物流相关的所有环节，包括打包、装柜、运输、清关及追踪 |
| Install | 安装 | 现场安装流程及协调工作，包括场地准备、安装记录等 |
| After Completion | 完工后阶段 | 安装完成后的阶段工作，包括项目总结、最终结算、质保资料提交及置留金确认与追踪等 |
