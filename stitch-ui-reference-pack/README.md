# Stitch UI Reference Pack

This folder is a reference package for using Stitch ideate/redesign with Project_R.

Use `DESIGN.md` as the primary upload/paste file. The `frontend-ui-code/` folder contains copied reference files from the current Project_R frontend so Stitch can see the real feature surface and avoid deleting important controls.

Suggested Stitch usage:

1. Paste or upload `DESIGN.md`.
2. Upload the `frontend-ui-code/` folder as code reference if Stitch allows it.
3. Add a note: "Optimize the layout and visual hierarchy, but preserve every functional control group listed in DESIGN.md."

This package is not wired into the app build. It is a design/reference bundle only.
