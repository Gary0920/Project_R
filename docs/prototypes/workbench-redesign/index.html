Project_R 工作台 UI 重设计原型

本原型用于确认新的整体风格、布局、通知系统、深浅色主题和客户情报面板设计方向。
所有数据均为静态假数据，不接真实 API，可直接用浏览器打开。

变体 A · 推荐

经典三栏 · 对话优先

继承 Proma 布局节奏，左侧导航 + 中间对话 + 右侧上下文面板。
柔和浅绿强调色，卡片层次分明，适合长时间办公。
客户情报收纳于右侧面板，与资料文件并列。

绿色强调色 260px 侧栏 右侧情报面板 通知下拉
变体 B · 探索

高密度 · 情报主舞台

更大胆的信息架构：客户情报从侧边窄条解放到主舞台全宽展示。
紧凑侧栏、扁平输入区、右侧滑入通知抽屉。
更适合"客户情报是核心业务视图"的定位。

靛蓝强调色 200px 紧凑侧栏 情报全宽覆盖 通知侧抽屉
操作提示： 在每个变体页面左下角有「主题 / 风格 / 面板」切换控件，可实时预览浅色/深色模式。
变体页面内所有按钮、Tab、下拉菜单均有交互（纯前端 JS），可点击体验。
