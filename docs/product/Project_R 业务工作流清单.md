# Project_R 业务工作流清单

- **来源**：`references/250711 Template [Project Check List].xmind`（公司项目全生命周期清单）
- **业务领域**：门窗幕墙行业项目管理
- **用途**：作为 Skill 开发的优先级清单与候选业务库

---

## 文档说明

本清单基于公司现有的项目 Check List 拆解，每个工作流节点都是一个潜在的 Skill 候选。

**优先级标记**：
- 🟢 **高** — 高频使用、规则明确、容易标准化
- 🟡 **中** — 中频使用或需要更多业务规则梳理
- ⚪ **低** — 低频或个性化较强，暂缓开发

**实现状态**：
- ⏸️ 未启动 / 🚧 设计中 / ✅ 已实现
